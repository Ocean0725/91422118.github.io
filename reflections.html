網頁設計專題 - 學習心得

在這個網頁製作專案中，我學到了如何利用 HTML5 與 CSS3 來建構多頁面（Multi-page）網站。在規劃高雄建築地標主題時，不僅需要思考首頁（index.html）與各個分頁之間的**超連結跳轉邏輯**，更要在 CSS 視覺設計上與地標本身的現代流線感相呼應。

最大的收穫是練習了 **CSS RWD（響應式網頁設計）** 的 Grid 與 Flexbox 佈局，讓網頁在手機與電腦螢幕上都能有良好的易讀性。此外，在心得頁中使用了毛玻璃（Glassmorphism）純文字背景特效，省去了大圖的沉重感，反而更突顯文字內容。

最後，將檔案推送到 GitHub 並開啟 **GitHub Pages** 服務，讓我初次體驗到網頁從在地端開發（Localhost）到實際「上線公諸於世」的完整工程流程，對於未來操作易課平台等線上管理系統也更有信心！
